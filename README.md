
Project Overview
This project performs a comprehensive Data Analysis on Amazon sales data to uncover insights regarding product performance, pricing strategies, and customer satisfaction. The goal is to provide actionable business recommendations based on data-driven evidence.
==> Phases of the Project
1. Data Cleaning & Preprocessing
Handling Missing Values: Identified and treated nulls to ensure data integrity.
Data Type Conversion: Converted prices and ratings from strings to numeric formats for analysis.
Outlier Detection: Identified unusual prices and abnormal ratings to clean the dataset from inconsistencies.
2. Descriptive Statistics
Category Distribution: Analyzed the number of products per category using groupby and value_counts.
Price & Rating Summary: Calculated mean, median, and standard deviation for key metrics.
3. Data Visualization
Created meaningful plots to answer business questions:
Correlation Heatmap: To find relationships between price, discount, and rating.
Top-rated Products: Visualizing products with the highest customer satisfaction.
Most Reviewed Products: Identifying the most popular items based on engagement.
==> Key Business Insights
Price vs. Rating: Discovered if more expensive products receive better ratings.
Discount Impact: Analyzed if higher discounts actually lead to increased popularity or better reviews.
==> Technologies Used
Python (Pandas, NumPy)
Visualization: Matplotlib & Seaborn
Environment: VS Code / Jupyter Notebook
