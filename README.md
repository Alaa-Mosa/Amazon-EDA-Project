##Amazon Analysis Project

A Data Science / Machine Learning project analyzing Amazon products and sales data.  

##About
This project focuses on exploring Amazon datasets, building predictive models, and visualizing insights.

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-Learn
- TensorFlow / PyTorch (if using ML models)

##  Features
- Data Cleaning and Analysis
- Sales Prediction Model
- Product Recommendation System
- Visualizations of top-selling products
