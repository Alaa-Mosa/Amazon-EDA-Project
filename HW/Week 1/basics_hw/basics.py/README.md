AskMe – Command Line Anonymous Q&A System

## Motivation

The main motivation behind this project was to practice building a complete Python application using Object-Oriented Programming. The project focuses on understanding class design, data persistence, input validation, and modular project structure without using databases or external libraries.

## Why This Project?

This project was built to simulate a real-world question-and-answer platform similar to Ask.fm but implemented as a Command Line Interface application. The goal was to focus on backend logic, clean architecture, and proper separation of responsibilities.

## Problem Statement

The project allows users to ask questions anonymously or publicly, manage multiple users, and store questions and answers persistently across application runs without relying on a database system.
